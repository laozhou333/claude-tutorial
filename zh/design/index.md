# 设计资源库

54 个顶级网站的完整设计系统，来自 [awesome-design-md](https://github.com/VoltAgent/awesome-design-md) 项目。

## 这是什么？

每个 DESIGN.md 文件都完整记录了一个知名网站的设计系统，包括：

- **视觉主题** — 整体风格、氛围、设计哲学
- **色彩系统** — 完整的调色板和使用规则
- **字体排版** — 字体选择、大小、行高、间距
- **组件样式** — 按钮、卡片、表单、导航等
- **动效系统** — 动画时长、缓动函数、过渡效果
- **响应式策略** — 断点、布局适配方案

## 如何使用？

将任意 DESIGN.md 复制到你的项目根目录，Claude Code 会自动读取并按照该设计系统生成 UI 代码。

```bash
# 例如：用 Stripe 的设计风格构建你的项目
cp DESIGN.md /your-project/DESIGN.md
# 然后让 Claude Code 按照这个设计系统来写代码
```

详细使用方法请看 [如何使用 →](./how-to-use)
